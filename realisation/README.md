# Mini Quiz PHP

## Description
Application PHP simple de quiz sans base de données.

## Fonctionnalités
- Questions à choix unique
- Calcul du score
- Affichage du résultat

## Contraintes respectées
- Pas de POO
- Pas de base de données
- 100% PHP local

## Technologies
- PHP
- HTML

## Lancement
1. Copier le dossier dans htdocs
2. Démarrer Apache (XAMPP)
3. Ouvrir http://localhost/quiz-php
